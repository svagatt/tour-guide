## Walking tour Planner

Ever been in a small town with a day or two to explore and wasted time on planning where to go? If yeah, this is the right place.

For now, clone the repo, then follow the steps,  
```bash
npm install
```
Create a .env file
```
touch .env.local
```
Add your anthropic api key `ANTHROPIC_API_KEY=your_key` to the env file.

Run the command, 

```bash
npm run dev
```
should be running on `localhost:3000`
